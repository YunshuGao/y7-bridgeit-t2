# Y7 Technology · Bridge It · Term 2 2026

Student-facing teaching hub for Year 7 Technology (Mandatory), Term 2 2026, at Arthur Phillip High School.

**Live site:** https://yunshugao.github.io/y7-bridgeit-t2/

## What's inside

- `index.html` — hub landing (Term 1 recap + Term 2 pointer)
- `term2.html` — Bridge It unit dashboard with all 22 lessons grouped by phase
- `lessons/l01.html` → `l22.html` — 22 × 75-minute lesson pages (I Do / We Do / You Do / Exit)
- `exemplars/` — truss-types reference, Design Folio exemplar, annotated evaluation sample
- `assets/style.css` — shared stylesheet (Inter body, Navy + Teal + phase colour coding)

## Unit overview

**Bridge It — Design, Build & Test an Engineered Bridge**

- 11 weeks · 22 lessons × 75 min · pair work
- Popsicle stick truss bridges · PVA + supervised hot glue
- 350 mm span · hanging load test to failure
- One summative Assessment Task (first for the cohort) split into:
  - Part A — Design Folio (40%) due Week 7
  - Part B — Bridge Build (30%) due Week 10
  - Part C — Load Test + Evaluation (30%) due Week 11

NSW NESA outcomes: TE4-1DP, TE4-2DP, TE4-3DP, TE4-8EN, TE4-9MA, TE4-10TS.

## Built with

Plain HTML + CSS. Python generators for lessons (`_build/build_lessons.py`, not in repo). No JS frameworks.

The stylesheet is deliberately readable — students can inspect it during L01.
